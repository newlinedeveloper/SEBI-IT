### üîπ **1. Basic Definitions**

| Term                   | Meaning                                 |
| ---------------------- | --------------------------------------- |
| **Cost Price (CP)**    | The price at which an article is bought |
| **Selling Price (SP)** | The price at which an article is sold   |
| **Profit (Gain)**      | SP > CP ‚áí Profit = SP ‚Äì CP              |
| **Loss**               | CP > SP ‚áí Loss = CP ‚Äì SP                |

---

### üîπ **2. Key Formulas**

[
\text{Profit %} = \frac{(SP - CP)}{CP} √ó 100
]

[
\text{Loss %} = \frac{(CP - SP)}{CP} √ó 100
]

---

### üîπ **3. When % is Given**

| Condition          | Formula                     |
| ------------------ | --------------------------- |
| **SP** when profit | SP = CP √ó (1 + p/100)       |
| **SP** when loss   | SP = CP √ó (1 ‚àí p/100)       |
| **CP** when profit | CP = SP √ó (100 / (100 + p)) |
| **CP** when loss   | CP = SP √ó (100 / (100 ‚àí p)) |

### üîπ **4. Example (Basic)**

> A man buys a book for ‚Çπ400 and sells it for ‚Çπ480. Find profit %.

### üîπ **5. Marked Price (MP) and Discount**

| Formula                                 | Description                    |
| --------------------------------------- | ------------------------------ |
| SP = MP ‚àí Discount                      | Discount given on marked price |
| Discount % = (Discount / MP) √ó 100      |                                |
| Profit or Loss % = ((SP ‚àí CP)/CP) √ó 100 |                                |

**Example:**

### üîπ **6. Successive Profit or Loss**

If an article is sold twice at **x%** and **y%** profit (or loss),
